###### GOAL #####
VisionCore ships as a pre-built Ubuntu 22.04 image that 
users flash to an Orange Pi 5 (or run the install script 
on any other supported device). On first boot, the systemd
service starts automatically and the user opens a browser to 
visioncore.local to configure everything. No SSH required for 
normal use.
